# Jera Capital Calculator

This project was developed as part of the selection process for Jera Capital.

The objective was to create a investment calculator able to calculate:

- Final projected value for each scenario
- Comparative chart of month-by-month evolution
- Percentage difference between scenarios
- Applicable Income Tax and IOF (Tax on Financial Transactions)

The technical description and instructions to run it locally is indise each app:
- [UI](https://github.com/FelipeCortesRocha/jera-capital-calculator/blob/main/jera-capital-ui/README.md)
- [API](https://github.com/FelipeCortesRocha/jera-capital-calculator/blob/main/jera-capital-api/README.md)
